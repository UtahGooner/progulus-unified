The WARP SECTOR Variant aims at creating balanced starting setups for player counts that normally result in asymmetrical layouts.

LARGE WARP SPACE



Using the Large Warp Space, the five-player setup becomes as balanced as possible by eliminating 1 Inner Sector Zone and two Middle Sector Zone hexes from the game.

Using two or three Large Warp Spaces it is also possible to create tighter 2 and 3 player games by eliminating Zone I and Zone II hexes and creating more possible connections to the Zone III areas. This will likely require some more table space than the normal setup for these player counts, just so you know..

There are several starting layouts that may be created for 7 player games with the help of one Large and one Small Warp Space. These layouts are not completely symmetric, but should be more balanced than playing without the Warp Spaces in any case. Using Warp Spaces will also help in creating these larger games without necessarily needing to add extra hex tiles to the game.

The variant thread can be found here.


This file is a printable A3-size PDF for the Large Warp Space Player Aid.

Instructions for use:

1) Print at 100 % size on A3 paper
2) Mount on cardboard
3) Cut carefully along the hex outline (there is a faint line for this purpose) - remember to be careful while cutting, watch those fingers!
